# gc_engine
unified and high performance garbage collector 

A garbage collection engine written in c. 

##Goal 

This project is intended to implement and compare different garbage collection techniques : i) reference couting ii) mark and sweep, from scratch 

Planned implementation : 
1) Reference counting 
2) Mark and sweep 
3) Heap visualizer 
4) Runtime statistics 
5) Memory Leak detection 
6) Bench marking
